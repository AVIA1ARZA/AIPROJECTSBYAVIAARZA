## 🛠️ הפרויקטים במאגר

### 1. 📊 SQL Insurance Simulator & AI Mentor
מערכת חכמה המשלבת ממשק משתמש מתקדם (React + Vite) עם מודל ה-API של Gemini. האפליקציה משמשת כסימולטור ומנטור אישי לתרגול וניתוח שאילתות SQL בעולמות הביטוח, תוך ביצוע אופטימיזציה של ביצועים ומערכות לגאסי (AS400/DB2).

* **טכנולוגיות:** React, TypeScript, Vite, `@google/genai`, Firebase.
* **📂 מעבר לקוד הפרויקט:** [צפייה בתיקיית SQL Insurance](./SQL_Insurance)

## 🚀 כיצד להריץ את הפרויקטים באופן מקומי?

לכל פרויקט יש קובץ הגדרות משלו. יש להיכנס לתיקיית הפרויקט הרצוי, להגדיר את מפתח ה-API בקובץ `.env` מקומי (לפי פורמט ה-`.env.example` המצורף בכל תיקייה), ולהריץ את פקודות ההתקנה המתאימות (`npm install` או `pip install`).
